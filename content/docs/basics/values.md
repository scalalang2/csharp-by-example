## Values

C#은 string, char, boolean, int 등 다양한 타입을 지원합니다.

<--->

```csharp
Console.WriteLine(1 + 2);
Console.WriteLine('a');
Console.WriteLine(true);
Console.WriteLine(1 + 1 == 2);
Console.WriteLine(2.0/4.0);
Console.WriteLine(2.0F/4.0F);
Console.WriteLine("Hello");
```

```bash
$ dotnet run
3
a
True

0.5
```
